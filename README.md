Repositorio donde documento cómo resolví distintos ejercicios de LeetCode, aplicando herramientas matemáticas de análisis asintótico.

Cada carpeta contiene un `Main` con la plantilla de la función esperada y un comentario con la descripción del problema. Cuando el ejercicio lo requiere, dentro de la misma subcarpeta también se agregan clases adicionales.

Además, dejo algunos intentos fallidos junto con una explicación de por qué no eran soluciones óptimas.
